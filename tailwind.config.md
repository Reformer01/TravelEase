# Tailwind CSS Configuration

This file configures Tailwind CSS for the VoyageFlow application with custom theming and design system.

## Configuration Details:

### Content Sources
- Pages: `./src/pages/**/*.{js,ts,jsx,tsx,mdx}`
- Components: `./src/components/**/*.{js,ts,jsx,tsx,mdx}`
- App: `./src/app/**/*.{js,ts,jsx,tsx,mdx}`

### Design System
- **Typography**: Public Sans font family for body, display, headline, and code
- **Color Scheme**: Custom color palette with CSS variables for theming
  - Primary brand color: #ec5b13 (orange)
  - Light background: #f8f6f6
  - Dark background: #221610
- **Border Radius**: Consistent radius scale (sm, md, lg, xl, full)
- **Dark Mode**: Class-based dark mode support

### Custom Components
- Sidebar color system for navigation
- Chart color palette for data visualization
- Accordion animations for UI components

### Plugins
- `tailwindcss-animate` for smooth animations
